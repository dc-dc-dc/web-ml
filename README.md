# Web-ml

This is a WIP project for inference only web-ml showcases.

## Op Support

### Memory Ops
- reshape
- flatten

### Unary Ops
- exp / exp2
- log / log2
- sqrt
- pow
- round
- ceil
- floor
- neg
- sin / sinh / asin
- cos / cosh / acos
- tan / tanh / atan

### Binary Ops
- add
- sub
- mul
- div
- max
- min

## Dev run

Currently must install locally

Install dependencies
```shell
npm install
```

Run tests
```shell
npm run test
```
